## 🔐 PROCESSUS DE CONNEXION PAR MÉTHODE

```
2. Écran de login s'affiche
3. Saisir email: test.artist@wmahub.ua
4. Saisir password: TestArtist2024!
5. Taper "Connexion"
6. Sélectionner le rôle si demandé
7. Attendre sync données (2-3 sec)
8. Dashboard s'affiche ✅

VALIDATION:
✅ Pas d'erreur d'authentification
✅ Redirection fluide
✅ Données chargées
✅ Notifications reçues
✅ Interface responsive
```

### Méthode 2: Google Sign-In (Recommandé)

```
ÉTAPES:
1. Taper "Sign In with Google"
2. Redirection vers page Google auth
3. Saisir: test.artist@wmahub.ua (ou Google account)
4. Saisir password Google
5. Confirmer les permissions
6. Retour à l'app
7. Sélectionner rôle si nouveau
8. Dashboard s'affiche ✅

PERMISSIONS DEMANDÉES:
✅ Email
✅ Profile (nom, avatar)
✅ Contact email

DONNÉES PRÉ-REMPLIES:
├─ Email: auto-rempli
├─ Name: auto-rempli
└─ Avatar: auto-rempli (si disponible)
```

### Méthode 3: Apple Sign-In (Obligatoire pour App Store)

```
ÉTAPES (iOS):
1. Taper "Sign In with Apple"
2. Sheet d'authentification Apple s'ouvre
3. Vérifier ID Apple
4. Sélectionner: "Share My Email" ou "Hide My Email"
5. Si Hide My Email: relayed email générée (xyz@privaterelay.appleid.com)
6. Taper "Continue" ou "Sign In"
7. Authentification biométrique (Face ID/Touch ID)
8. Revenir à l'app
9. Sélectionner rôle si nouveau
10. Dashboard s'affiche ✅

OPTIONS APPLE:
├─ Share My Email: Email visible à WMA Hub
└─ Hide My Email: Email caché (Apple génère relay)

PERMISSIONS:
✅ Aucune donnée supplémentaire demandée
✅ Pas d'accès à contacts/calendrier
✅ Nom optionnel (l'utilisateur peut l'omettre)
```
